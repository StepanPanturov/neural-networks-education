# Радиально-базисные нейронные сети: программно-учебные материалы

Этот проект представляет собой программно-учебные материалы для изучения радиально-базисных нейронных сетей (РБНС). Проект разработан с целью наглядной демонстрации принципов работы, обучения и применения РБНС для решения задач аппроксимации функций и классификации.

## Описание

Радиально-базисные нейронные сети (Radial Basis Function Networks, RBFN) - это особый тип нейронных сетей, использующих радиальные базисные функции в качестве функций активации. Они отличаются простой трёхслойной архитектурой и обладают рядом преимуществ для решения задач аппроксимации и интерполяции данных.

Данный проект предоставляет комплексную платформу для изучения РБНС, включающую:

**Образовательные возможности:**
- Интерактивное изучение принципов построения и работы РБНС
- Пошаговую визуализацию процесса обучения с элементами интерактива
- Наглядную демонстрацию различных типов радиальных базисных функций
- Исследование влияния гиперпараметров на качество аппроксимации

**Экспериментальные инструменты:**
- Систематическое тестирование различных конфигураций сети
- Сравнительный анализ эффективности на разнообразных наборах данных
- Метрики оценки качества работы и их визуализация
- Расширенные эксперименты для углубленного изучения

## Архитектура проекта

### Основные исполняемые модули

**`main.py`** - центральная точка входа для демонстрации всех возможностей проекта. Запускает последовательность демонстраций, охватывающих ключевые аспекты работы РБНС.

**`experiment_with_rbf.py`** - специализированный модуль для проведения систематических экспериментов с различными параметрами РБНС. Позволяет автоматизировать процесс тестирования и анализа результатов.

**`rbfn_class.py`** - ядро проекта, содержащее полную реализацию класса RBFNetwork с функциональностью создания, обучения, визуализации и тестирования радиально-базисных нейронных сетей.

### Модуль генерации данных (`generate_data/`)

**`generate_demo_data.py`** - создание базовых демонстрационных наборов данных для изучения основных принципов работы РБНС. Включает простые одномерные и двумерные функции.

**`generate_advanced_demo_data.py`** - генерация сложных и многомерных наборов данных для проведения расширенных экспериментов. Предназначен для углубленного изучения возможностей РБНС на нетривиальных задачах.

### Демонстрационные и вспомогательные функции (`helping_funcs/`)

**`demo_rbf_formation.py`** - наглядная демонстрация процесса формирования радиальных базисных функций, их размещения в пространстве признаков и влияния на итоговую аппроксимацию.

**`demonstrate_rbf_types.py`** - сравнительный анализ различных типов радиальных базисных функций (Гауссова, мультиквадратичная, обратная мультиквадратичная и др.) и их влияния на качество обучения.

**`demo_training_process.py`** - визуализация пошагового процесса обучения РБНС с отображением изменения весов, центров и ширины функций.

**`demo_training_process_interactive.py`** - интерактивная версия демонстрации обучения с возможностью вмешательства пользователя в процесс и изменения параметров в реальном времени.

**`demo_experiment_neurons_width.py`** - систематическое исследование влияния количества нейронов и ширины радиальных функций на качество аппроксимации и обобщающую способность сети.

**`demo_dataset_comparison.py`** - сравнительный анализ эффективности РБНС на различных типах данных: линейных, нелинейных, с шумом, многомерных.

**`demo_advanced_experiments.py`** - проведение сложных экспериментов, включающих оптимизацию гиперпараметров, анализ устойчивости к переобучению и тестирование на реальных данных.

**`plot_dataset_results.py`** - специализированные функции для создания детальных графических представлений результатов аппроксимации с различными стилями визуализации.

### Модуль визуализации (`visualization/`)

**`visualize_experiment_results.py`** - комплексная визуализация результатов экспериментов с созданием сводных графиков, диаграмм сравнения и интерактивных отчетов.

**`visualize_metrics.py`** - специализированные инструменты для отображения метрик качества работы сети: MSE, MAE, R², матрицы ошибок для задач классификации.

### Теоретическая документация

**`Theory.md`** - подробное теоретическое описание радиально-базисных нейронных сетей, математическое обоснование алгоритмов, примеры применения и сравнение с другими методами машинного обучения.

## Основные возможности класса RBFNetwork

Реализованный класс RBFNetwork предоставляет полный набор функций для работы с РБНС:

- **Гибкая конфигурация**: настройка количества нейронов, типа базисной функции, методов инициализации центров
- **Обучение**: реализация различных алгоритмов обучения с контролем сходимости
- **Визуализация**: создание статических графиков и анимаций процесса обучения
- **Анализ**: вычисление метрик качества и анализ поведения сети
- **Предсказание**: эффективное получение прогнозов на новых данных
